# WallpaperDuck
一个随机壁纸小工具

就像万花筒中的图案， 这个软件有着无限的可能性， 每一次切换都是一场奇妙的冒险
## 使用
点击这里下载
https://github.com/cnhkbbs/WallpaperDuck/releases

## 壁纸类型
```
# 0 随机
# 1 4K
# 2 美女
# 3 美图
# 4 风景
# 5 小清新
# 6 动漫卡通
# 7 明星
# 8 动物
# 9 游戏
```
