# WallpaperDuck
一个随机壁纸小工具
## 使用
点击这里下载
https://github.com/cnhkbbs/WallpaperDuck/releases
### 自定义
你可以通过修改config.json来进行自定义
```
{
    "hitokoto":true,  #是否打开一言     true或false
    "biying":false,   #启用必应美图     true或false  （启用该选项会使category选项失效）
    "category":0      #壁纸类型         值见下文     （该选项只在biying选项为flase时可用）
}
```

category可用值
```
# 0 随机
# 1 4K
# 2 美女
# 3 美图
# 4 风景
# 5 小清新
# 6 动漫卡通
# 7 明星
# 8 动物
# 9 游戏
```
